# 😊 NeuralFace – AI Real-Time Emotion Detection System

NeuralFace is an AI-powered real-time facial emotion recognition web application built using **Python, Flask, OpenCV, DeepFace, and Face-API.js**. It detects human emotions from both **live webcam feeds** and **uploaded images**, displaying results through an interactive analytics dashboard with confidence scores and visualizations.

Install the required libraries:

```bash

pip install numpy
pip install pillow
pip install deepface
pip install tensorflow
```

Or

```bash
pip install -r requirements.txt
```

# Installation

Clone the repository

```bash
git clone https://github.com/yourusername/NeuralFace.git
```

Move to project folder

```bash
cd NeuralFace
```

Install dependencies

```bash
pip install -r requirements.txt
```

Run the application

```bash
python app.py
```

Open your browser

```
http://127.0.0.1:5000
```
